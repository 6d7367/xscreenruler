# xscreenruler
Simple screen ruler for x11. Only xlib in dependencies!

![xscreenruler.png](https://github.com/6d7367/xscreenruler/blob/master/xscreenruler.png)

## DIRECTION
```
ctrl+n - north
ctrl+s - south
ctrl+w - west
ctrl+e - east
```

## SIZE
```
shift+d - default size
shift+s - small size
shift+m - medium size
shift+t - tall size
shift+f - fullscreen
```

## MOVEMENT
```
arrows (up, down, left, right)
ctrl+arrows - force


ESC or ctrl+q - exit
```
